# Delhi Meerut Cargo

Delhi Meerut Cargo is a full-stack logistics and transportation management system designed to simplify Proof of Delivery (POD) management and customer operations for transport companies.

The application enables secure user authentication with role-based access control for Admin, Staff, and Client accounts. Admins can approve new client registrations, manage users, and oversee the system, while Staff members can upload and manage PODs. Clients can securely search and view their PODs using the LR (Lorry Receipt) number.

## Features

* 🔐 JWT-based Authentication & Authorization
* 👥 Role-Based Access Control (Admin, Staff, Client)
* ✅ Admin Approval System for Client Accounts
* 📤 Upload PODs (Images & PDFs)
* 🔍 Search POD by LR Number
* 👁️ View POD in Browser
* 📥 Download POD with LR Number as Filename
* 🗑️ Delete POD (Admin & Staff Only)
* ☁️ Cloudinary Integration for File Storage
* 📱 Responsive Dashboard using React & Tailwind CSS
* 🔒 Protected Routes & Secure APIs

## Purpose

The primary goal of this project is to digitize POD management, reduce paperwork, improve accessibility, and provide a secure platform for logistics companies to manage Proof of Delivery records efficiently.

This project demonstrates practical implementation of authentication, authorization, file uploads, cloud storage, protected APIs, and full-stack application development.
